    Estructuras o registros.

    Una estructura es un conjunto de datos que no son necesariamente del mismo tipo. No hay limitaciones en el tipo ni cantidad de variables que pueda contener una estructura, la única limitación es que una estructura no puede contenerse a sí misma como miembro.
    Una estructura se declara con la palabra reservada struct y se declara utilizando los mismos pasos necesarios para utilizar cualquier variable. Primero, se debe declarar el registro y a continuación se asignan valores a los miembros o elementos individuales del registro o estructura.

    Estructuras de datos homogéneas y heterogéneas

    Los registros (estructuras) y los arrays son tipos de datos estructurados. La diferencia entre estos dos tipos de estructuras de datos son los tipos de elementos que ellos contienen.
    • Un array es una estructura de datos homogénea, que significa que cada uno de sus componentes deben ser del mismo tipo.
    • Un registro es una estructura de datos heterogénea, que significa que cada uno de sus componentes pueden ser de tipos de datos diferentes
    • Borrar elementos
    Por consiguiente, un array de registros es una estructura de datos cuyos elementos son de los mismos tipos heterogéneos.

    1.- El tipo numérico es el conjunto de los valores numéricos. Estos pueden representarse en dos formas distintas:

    1. Tipo numérico entero (integer)

    2. Tipo numérico real (real)

    2.- Los nombres de las variables elegidas para el algoritmo o el programa deben ser significativos y tener relación con el objeto que representan, como pueden ser los casos siguientes:

    1. NOMBRE para representar nombres de personas

    2. PRECIOS para representar los precios de diferentes artículos

    3. NOTAS para representar las notas de una clase

    3.- Cada expresión toma un valor que se determina tomando los valores de las variables y constantes implicadas y la ejecución de las operaciones indicadas. Una expresión consta de operandos y operadores. Según sea el tipo de objetos que manipulan, las expresiones se clasifican en:

    1. Aritméticas

    2. Relacionales

    3. Lógicas

    4. Carácter
